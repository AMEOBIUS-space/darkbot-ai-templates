# Crypto Trading Bot Template — Arbitrage, RSI, Multi-Exchange

Production-ready crypto trading bot:
- Multi-exchange price monitoring (Binance, OKX, Bybit)
- Arbitrage detection (cross-exchange spread)
- RSI technical indicator signals
- Signal-based order execution
- Candlestick data (klines) fetching
- Configurable thresholds (min spread, signal strength)
- Order tracking and position management
- Async main loop with rate limiting

## Quick Start
```bash
pip install -r requirements.txt
python bot.py  # Demo mode (price check + RSI signal)
# For live trading: add API keys to config
```
