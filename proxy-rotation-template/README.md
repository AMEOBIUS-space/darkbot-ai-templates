# Proxy Rotation Manager — Pool, Health Checks, Auto-Failover

Production-ready proxy rotation with:
- Round-robin and random rotation
- Concurrent health checks
- Auto-failover (mark unhealthy after N fails)
- Latency tracking
- File-based proxy list loading
- SOCKS5/SOCKS4/HTTP support
- Pool statistics

## Quick Start
```bash
pip install -r requirements.txt
python rotator.py
```
