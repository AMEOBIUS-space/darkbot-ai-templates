# Telegram Channel Scraper — Keyword Monitor, Analytics, Export

Production-ready Telegram scraper with:
- Multi-channel scraping (Telethon)
- Keyword filtering with match tracking
- Message metadata (views, reactions, media, sender)
- Configurable time window (days_back)
- JSON and CSV export
- Analytics dashboard (total messages, views, reactions, keyword counts)
- Async for speed

## Quick Start
```bash
pip install -r requirements.txt
# Set api_id, api_hash, session in scraper.py
python scraper.py
```
