# Captcha Solver Template — reCAPTCHA, hCaptcha, Turnstile, Yandex

Production-ready multi-provider captcha solver:
- reCAPTCHA v2 (2Captcha API)
- hCaptcha (CapSolver API)
- Cloudflare Turnstile (CapSolver API)
- Yandex SmartCaptcha (CDP checkbox approach)
- Async with polling
- Fallback between providers
- Elapsed time tracking

## Quick Start
```bash
pip install -r requirements.txt
export TWOCAPTCHA_KEY=your_key
export CAPSOLVER_KEY=your_key
python solver.py
```
