# Security Policy

## Reporting a Vulnerability

Found a security issue in a template? Contact us privately:

1. Telegram: [@darkbot_ai_bot](https://t.me/darkbot_ai_bot)
2. GitHub: Create a private security advisory

## Response Time

- Acknowledgment: 24 hours
- Fix: 48-72 hours depending on severity

## Scope

Security vulnerabilities in:
- Template code (Python, Solidity, JavaScript)
- Dependencies
- Configuration files

## Out of Scope

- Issues in third-party libraries (report upstream)
- Social engineering
- DoS attacks

## Safe Harbor

We won't take legal action against security researchers who:
- Act in good faith
- Don't access/modify user data
- Report vulnerabilities responsibly
