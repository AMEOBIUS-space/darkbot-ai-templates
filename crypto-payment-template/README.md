# Crypto Payment Gateway Template — BTC, USDT, ETH, XMR

Production-ready crypto payment gateway for digital products:
- Multi-currency (BTC, USDT, ETH, XMR)
- Invoice creation with timeout
- BTC payment verification (blockchain API)
- XMR payment verification (Monero RPC)
- Webhook notifications
- FastAPI integration (REST API)
- Callback system for paid invoices
- Invoice status tracking

## Quick Start
```bash
pip install -r requirements.txt
python gateway.py  # Demo mode
# Or use create_api() for FastAPI server
```
