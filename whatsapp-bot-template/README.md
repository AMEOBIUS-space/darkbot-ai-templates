# WhatsApp Bot Template — Baileys, Commands, Rate Limiting, Media

Production-ready WhatsApp bot with:
- Baileys integration (no official API needed)
- Command handlers (/start, /help, /info)
- Rate limiting per user
- Blocked numbers list
- Admin numbers
- Media sending (images, documents)
- Group message support
- Python orchestration + Node.js Baileys layer

## Quick Start
```bash
pip install -r requirements.txt
npm install @whiskeysockets/baileys @hapi/boom
python bot.py          # Python orchestration
node wa_node.js        # WhatsApp connection (shows QR)
```
