# Discord Bot Template — Slash Commands, Moderation, Music, Polls

Production-ready Discord bot with:
- Slash commands (synced automatically)
- Welcome messages
- User info embeds
- Message purge (moderation)
- Music queue system
- Poll creation with reactions
- Permission checks

## Quick Start
```bash
pip install -r requirements.txt
export DISCORD_TOKEN=your_token
python bot.py
```
