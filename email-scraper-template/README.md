# Email Scraper & Validator — Extract, Validate, Export

Production-ready email scraper with:
- Recursive website crawling (configurable depth)
- Same-domain link following
- Email extraction via regex
- Deduplication
- Basic email validation
- CSV and JSON export
- Async (aiohttp) for speed

## Quick Start
```bash
pip install -r requirements.txt
python scraper.py  # Scrapes example.com as demo
```
