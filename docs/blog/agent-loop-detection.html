Detecting and Breaking AI Agent Reasoning Loops

MCP Agents Python

2026-07-09

You have seen it: the agent reads a file, runs a search, reads the same file again, runs the same search, and repeats. Each iteration burns tokens. Twenty calls later, the context window is full, the task is unfinished, and the user has paid for a loop.

Loop detection is the difference between an agent that costs $0.05 per task and one that costs $2.00 for the same result.

Types of Loops

1. Exact Repeat Loops

The agent calls the same tool with identical arguments. Easiest to detect.

# Agent calls: read_file("config.yaml"), read_file("config.yaml"), read_file("config.yaml")
# Same tool, same args, same result. Textbook loop.

2. Semantic Loops

The agent calls different tools but with equivalent intent.

# Agent calls: search_files("import os"), search_files("import os$")
# Different regex, same goal. Harder to catch.

3. Spiral Loops

The agent cycles through a fixed set of actions that never converge.

# read_file → patch → test → read_file → patch → test → ...
# Each call is different, but the PATTERN repeats.

4. Context Loops

The agent keeps asking for clarification, the user gives the same answer, and the agent asks again.

Detection Implementation

Here is a loop detector that catches all four types. It tracks tool calls, their arguments, and the sequence pattern:

import hashlib
import json
from collections import deque
from dataclasses import dataclass, field

@dataclass
class ToolCall:
    name: str
    args: dict
    result_hash: str = ""

@dataclass 
class LoopDetector:
    window_size: int = 10
    repeat_threshold: int = 3
    pattern_threshold: int = 2
    calls: deque = field(default_factory=lambda: deque(maxlen=50))
    
    def record(self, name: str, args: dict, result: str = "") -> dict:
        result_hash = hashlib.md5(result.encode()).hexdigest()[:8] if result else ""
        call = ToolCall(name=name, args=args, result_hash=result_hash)
        self.calls.append(call)
        return self.check()
    
    def _call_signature(self, call: ToolCall) -> str:
        """Normalize args to catch semantic duplicates."""
        # Sort keys, stringify values
        normalized = json.dumps(call.args, sort_keys=True, default=str)
        return f"{call.name}:{hashlib.md5(normalized.encode()).hexdigest()[:8]}"
    
    def check(self) -> dict:
        if len(self.calls) < 3:
            return {"loop": False}
        
        recent = list(self.calls)
        
        # Check 1: Exact repeats (same signature 3+ times in window)
        window = recent[-self.window_size:]
        signatures = [self._call_signature(c) for c in window]
        for sig in set(signatures):
            count = signatures.count(sig)
            if count >= self.repeat_threshold:
                return {
                    "loop": True,
                    "type": "exact_repeat",
                    "tool": sig.split(":")[0],
                    "count": count,
                    "message": f"Tool '{sig.split(':')[0]}' called {count} times with same args"
                }
        
        # Check 2: Sequence pattern (A-B-C-A-B-C)
        for pattern_len in range(2, 6):
            if len(recent) >= pattern_len * self.pattern_threshold:
                pattern = signatures[-pattern_len:]
                repeats = 0
                for i in range(len(signatures) - pattern_len, -1, -pattern_len):
                    if signatures[i:i+pattern_len] == pattern:
                        repeats += 1
                    else:
                        break
                if repeats >= self.pattern_threshold:
                    return {
                        "loop": True,
                        "type": "sequence_pattern",
                        "pattern": pattern,
                        "repeats": repeats,
                        "message": f"Sequence of {pattern_len} tools repeated {repeats} times"
                    }
        
        # Check 3: Same result hash (different calls, same output)
        result_hashes = [c.result_hash for c in window if c.result_hash]
        for rh in set(result_hashes):
            if result_hashes.count(rh) >= 3:
                return {
                    "loop": True,
                    "type": "same_result",
                    "result_hash": rh,
                    "count": result_hashes.count(rh),
                    "message": f"Same result returned {result_hashes.count(rh)} times"
                }
        
        return {"loop": False}

Breaking Loops

Once detected, you need a strategy to break out. Five approaches, from gentle to aggressive:

Level 1: Nudge (inject a hint)

def inject_nudge(loop_info: dict) -> str:
    if loop_info["type"] == "exact_repeat":
        return (f"[LOOP DETECTED] You have called {loop_info['tool']} "
                f"{loop_info['count']} times with the same arguments. "
                f"The result will not change. Try a different approach.")
    elif loop_info["type"] == "sequence_pattern":
        return (f"[LOOP DETECTED] Your last {len(loop_info['pattern'])} tool calls "
                f"have repeated {loop_info['repeats']} times. "
                f"Summarize what you know and identify what is still missing.")
    return ""

Level 2: Force Reflection

Inject a user message that forces the agent to stop and think:

REFLECTION_PROMPT = """STOP. You are in a loop. Before making another tool call:
1. List what you have already tried
2. State what information you still need
3. Explain why your previous approach did not work
4. Propose a DIFFERENT approach
Do not repeat any tool call you have made in the last 5 turns."""

Level 3: Tool Restriction

Temporarily remove the looping tool from the available tools:

def restrict_looping_tool(loop_info: dict, available_tools: list) -> list:
    if loop_info["type"] == "exact_repeat":
        looping_tool = loop_info["tool"]
        return [t for t in available_tools if t != looping_tool]
    return available_tools

Level 4: Context Pruning

Remove the looping tool calls from conversation history, keeping only the first and last:

def prune_loop_turns(messages: list, loop_info: dict) -> list:
    """Remove duplicate tool calls from history, keep first + last."""
    if not loop_info.get("loop"):
        return messages
    
    pruned = []
    seen_signatures = {}
    
    for msg in messages:
        if msg.get("role") == "tool":
            sig = f"{msg.get('tool_call_id', '')}:{msg.get('content', '')[:50]}"
            if sig in seen_signatures:
                # Skip duplicate — replace with summary
                pruned.append({
                    "role": "system",
                    "content": f"[Previous duplicate tool call omitted]"
                })
                continue
            seen_signatures[sig] = len(pruned)
        pruned.append(msg)
    
    return pruned
