Why Zero Dependencies?

Every MCP server in this collection uses only the Python standard library. No pip installs, no Playwright, no Selenium, no requests, no npm. This means:

  • Install in seconds, not minutes
  • No supply chain risk from third-party packages
  • Works on any system with Python 3.8+
  • Minimal attack surface
  • Easy to audit — all code is pure stdlib
