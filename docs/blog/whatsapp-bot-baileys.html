WhatsApp Bot with Baileys

Build a WhatsApp bot without official Business API — using Baileys library.

Why Baileys?

Baileys connects to WhatsApp Web protocol directly. No official API key needed, no $50/month Business API fee.

1. Setup (Node.js)

npm install @whiskeysockets/baileys
node wa_node.js  # Shows QR code

2. Python Orchestration

class WhatsAppBot:
    def command(self, cmd):
        def decorator(func):
            self.handlers[cmd] = func
            return func
        return decorator

    async def process_message(self, msg):
        if msg.text.startswith("/"):
            cmd = msg.text[1:].split(" ")[0]
            if cmd in self.handlers:
                await self.handlers[cmd](msg)

3. Rate Limiting

def check_rate_limit(self, sender):
    now = time.time()
    self.tracker[sender] = [t for t in self.tracker[sender] if now-t < 60]
    return len(self.tracker[sender]) < 10
