Prompt Versioning for AI Agents: Track, Diff, and Roll Back

AI Agents Version Control Python MCP

You changed one line in your system prompt. Agent accuracy dropped 30%. When did it happen? Which version was better? What exactly changed? If you can't answer these questions in under 30 seconds, you need prompt versioning.

Most teams treat prompts like config files — edit, save, hope. This guide shows how to build a proper version control system for prompts in pure Python, with hashing, semantic diffing, rollback, and impact tracking.

Why Prompt Versioning Is Different from Git

Git tracks files. Prompts are runtime artifacts — they get composed from templates, injected with context, modified by skills, and sent to different models. You need to track:

AspectGit (files)Prompt Versioning (runtime)
What changesSource filesComposed prompts sent to models
WhenOn commitEvery API call
WhyDeveloper intentSkill injection, context addition, model routing
GranularityLines/filesSections, variables, few-shot examples
Rollback targetPrevious commitPrevious prompt that performed well

Step 1: The Version Store

Each prompt version gets a content hash, metadata, and tags. Identical prompts are deduplicated:

import hashlib
import json
import sqlite3
import time
from dataclasses import dataclass, field
from typing import Any

@dataclass
class PromptVersion:
    version_id: str           # UUID
    prompt_hash: str          # SHA-256 of content
    content: str
    model: str = ""
    tags: list[str] = field(default_factory=list)
    label: str = ""           # Human-readable label ("v2.1-cot")
    parent_hash: str = ""     # Previous version's hash
    metadata: dict[str, Any] = field(default_factory=dict)
    created_at: float = field(default_factory=time.time)
    is_pinned: bool = False   # Pinned = can't be auto-rolled-back


class PromptVersionStore:
    def __init__(self, db_path: str = ":memory:"):
        self.conn = sqlite3.connect(db_path)
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS prompt_versions (
                version_id TEXT PRIMARY KEY,
                prompt_hash TEXT UNIQUE NOT NULL,
                content TEXT NOT NULL,
                model TEXT DEFAULT '',
                tags TEXT DEFAULT '[]',
                label TEXT DEFAULT '',
                parent_hash TEXT DEFAULT '',
                metadata TEXT DEFAULT '{}',
                created_at REAL,
                is_pinned INTEGER DEFAULT 0
            )
        """)
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS prompt_impact (
                prompt_hash TEXT,
                metric_name TEXT,
                metric_value REAL,
                sample_count INTEGER,
                recorded_at REAL,
                PRIMARY KEY (prompt_hash, metric_name)
            )
        """)
        # FTS5 for full-text search across prompt content
        self.conn.execute("""
            CREATE VIRTUAL TABLE IF NOT EXISTS prompt_search
            USING fts5(content, content='prompt_versions', content_rowid='rowid')
        """)
        self.conn.commit()

    def save(self, content: str, model: str = "",
             tags: list[str] | None = None,
             label: str = "",
             parent_hash: str = "",
             metadata: dict | None = None) -> PromptVersion:
        import uuid

        prompt_hash = hashlib.sha256(content.encode()).hexdigest()[:16]

        # Dedup: if exact content already stored, return existing
        existing = self.conn.execute(
            "SELECT version_id FROM prompt_versions WHERE prompt_hash = ?",
            (prompt_hash,)
        ).fetchone()

        if existing:
            return self.get_by_hash(prompt_hash)

        version = PromptVersion(
            version_id=uuid.uuid4().hex[:12],
            prompt_hash=prompt_hash,
            content=content,
            model=model,
            tags=tags or [],
            label=label,
            parent_hash=parent_hash,
            metadata=metadata or {},
        )

        self.conn.execute("""
            INSERT INTO prompt_versions
            (version_id, prompt_hash, content, model, tags, label,
             parent_hash, metadata, created_at, is_pinned)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, 0)
        """, (
            version.version_id, version.prompt_hash, version.content,
            version.model, json.dumps(version.tags), version.label,
            version.parent_hash, json.dumps(version.metadata),
            version.created_at,
        ))
        self.conn.commit()

        return version

    def get_by_hash(self, prompt_hash: str) -> PromptVersion | None:
        row = self.conn.execute(
            "SELECT * FROM prompt_versions WHERE prompt_hash = ?",
            (prompt_hash,)
        ).fetchone()
        return self._row_to_version(row) if row else None

    def list_versions(self, limit: int = 20,
                      tag: str | None = None) -> list[PromptVersion]:
        if tag:
            rows = self.conn.execute("""
                SELECT * FROM prompt_versions
                WHERE tags LIKE ?
                ORDER BY created_at DESC LIMIT ?
            """, (f'%"{tag}"%', limit)).fetchall()
        else:
            rows = self.conn.execute("""
                SELECT * FROM prompt_versions
                ORDER BY created_at DESC LIMIT ?
            """, (limit,)).fetchall()
        return [self._row_to_version(r) for r in rows]

    def _row_to_version(self, row) -> PromptVersion:
        return PromptVersion(
            version_id=row[0],
            prompt_hash=row[1],
            content=row[2],
            model=row[3],
            tags=json.loads(row[4]),
            label=row[5],
            parent_hash=row[6],
            metadata=json.loads(row[7]),
            created_at=row[8],
            is_pinned=bool(row[9]),
        )

Step 2: The Diff Engine

Three diff levels — choose the right one for the job:

Level 1: Token-Level Diff (Precise)

import difflib

class PromptDiffer:
    """Three-level diff engine for prompts."""

    def token_diff(self, old: str, new: str) -> list[dict]:
        """Character-level unified diff — catches every change."""
        differ = difflib.unified_diff(
            old.splitlines(keepends=True),
            new.splitlines(keepends=True),
            fromfile="old",
            tofile="new",
            n=2,
        )

        changes = []
        for line in differ:
            if line.startswith("+") and not line.startswith("+++"):
                changes.append({"type": "add", "content": line[1:].strip()})
            elif line.startswith("-") and not line.startswith("---"):
                changes.append({"type": "del", "content": line[1:].strip()})

        return changes

    def summarize_diff(self, changes: list[dict]) -> dict:
        """Aggregate diff stats."""
        additions = sum(1 for c in changes if c["type"] == "add")
        deletions = sum(1 for c in changes if c["type"] == "del")

        return {
            "additions": additions,
            "deletions": deletions,
            "total_changes": additions + deletions,
            "change_ratio": (additions + deletions) / max(len(changes), 1),
        }

Level 2: Section-Level Diff (Structured)

class PromptDiffer:
    # ... continued ...

    def section_diff(self, old: str, new: str) -> list[dict]:
        """Diff by markdown sections (## headers)."""

        old_sections = self._split_sections(old)
        new_sections = self._split_sections(new)

        results = []
        all_keys = list(dict.fromkeys(
            list(old_sections.keys()) + list(new_sections.keys())
        ))

        for key in all_keys:
            old_text = old_sections.get(key, "")
            new_text = new_sections.get(key, "")

            if old_text and not new_text:
                results.append({
                    "section": key, "status": "removed",
                    "old_lines": old_text.count("\n"),
                })
            elif new_text and not old_text:
                results.append({
                    "section": key, "status": "added",
                    "new_lines": new_text.count("\n"),
                })
            elif old_text != new_text:
                results.append({
                    "section": key, "status": "modified",
                    "changes": self.token_diff(old_text, new_text),
                })

        return results

    def _split_sections(self, text: str) -> dict[str, str]:
        """Split prompt into sections by ## headers."""
        sections: dict[str, str] = {"_header": ""}
        current = "_header"
        lines = text.split("\n")

        for line in lines:
            if line.strip().startswith("##"):
                current = line.strip().lstrip("#").strip()
                sections[current] = line + "\n"
            else:
                sections[current] = sections.get(current, "") + line + "\n"

        return sections

Level 3: Semantic Diff (Intent-Level)

class PromptDiffer:
    # ... continued ...

    def semantic_diff(self, old: str, new: str) -> dict:
        """High-level semantic comparison — what changed in intent?"""

        old_words = set(old.lower().split())
        new_words = set(new.lower().split())

        added_words = new_words - old_words
        removed_words = old_words - new_words

        # Categorize changes by type
        instruction_words = {"must", "should", "never", "always",
                             "do", "dont", "avoid", "ensure"}
        formatting_words = {"format", "json", "markdown", "xml",
                           "table", "list"}
        behavior_words = {"think", "reason", "explain", "summarize",
                         "analyze", "compare"}

        def categorize(word: str) -> str:
            if word in instruction_words:
                return "instruction"
            if word in formatting_words:
                return "formatting"
            if word in behavior_words:
                return "behavior"
            return "content"

        added_by_cat: dict[str, list[str]] = {}
        for w in added_words:
            cat = categorize(w)
            added_by_cat.setdefault(cat, []).append(w)

        removed_by_cat: dict[str, list[str]] = {}
        for w in removed_words:
            cat = categorize(w)
            removed_by_cat.setdefault(cat, []).append(w)

        return {
            "added_words": len(added_words),
            "removed_words": len(removed_words),
            "categories_added": {k: len(v) for k, v in added_by_cat.items()},
            "categories_removed": {k: len(v) for k, v in removed_by_cat.items()},
            "net_instruction_change": (
                len(added_by_cat.get("instruction", []))
                - len(removed_by_cat.get("instruction", []))
            ),
        }

Step 3: Impact Tracking

Every prompt version accumulates impact metrics. You see which version performs best, not just which is newest:

class ImpactTracker:
    """Track how each prompt version performs in production."""

    def __init__(self, store: PromptVersionStore):
        self.store = store

    def record(self, prompt_hash: str, metrics: dict[str, float]) -> None:
        """Record a single observation for a prompt version."""
        for name, value in metrics.items():
            # Upsert: accumulate running average
            row = self.store.conn.execute("""
                SELECT metric_value, sample_count FROM prompt_impact
                WHERE prompt_hash = ? AND metric_name = ?
            """, (prompt_hash, name)).fetchone()

            if row:
                old_avg, count = row
                new_count = count + 1
                new_avg = (old_avg * count + value) / new_count
                self.store.conn.execute("""
                    UPDATE prompt_impact
                    SET metric_value = ?, sample_count = ?, recorded_at = ?
                    WHERE prompt_hash = ? AND metric_name = ?
                """, (new_avg, new_count, time.time(), prompt_hash, name))
            else:
                self.store.conn.execute("""
                    INSERT INTO prompt_impact
                    (prompt_hash, metric_name, metric_value, sample_count, recorded_at)
                    VALUES (?, ?, ?, 1, ?)
                """, (prompt_hash, name, value, time.time()))

        self.store.conn.commit()

    def compare(self, hash_a: str, hash_b: str) -> dict:
        """Side-by-side comparison of two prompt versions."""
        metrics_a = self._get_metrics(hash_a)
        metrics_b = self._get_metrics(hash_b)

        all_keys = set(metrics_a.keys()) | set(metrics_b.keys())
        comparison = {}

        for key in all_keys:
            val_a = metrics_a.get(key, {})
            val_b = metrics_b.get(key, {})

            avg_a = val_a.get("value", 0)
            avg_b = val_b.get("value", 0)

            delta = avg_b - avg_a
            pct = (delta / avg_a * 100) if avg_a else 0

            comparison[key] = {
                "version_a": round(avg_a, 4),
                "version_b": round(avg_b, 4),
                "delta": round(delta, 4),
                "delta_pct": round(pct, 1),
                "samples_a": val_a.get("count", 0),
                "samples_b": val_b.get("count", 0),
            }

        return comparison

    def _get_metrics(self, prompt_hash: str) -> dict:
        rows = self.store.conn.execute("""
            SELECT metric_name, metric_value, sample_count
            FROM prompt_impact WHERE prompt_hash = ?
        """, (prompt_hash,)).fetchall()

        return {
            row[0]: {"value": row[1], "count": row[2]}
            for row in rows
        }

Step 4: Rollback

class PromptRollback:
    """Safe rollback to previous prompt versions."""

    def __init__(self, store: PromptVersionStore,
                 tracker: ImpactTracker):
        self.store = store
        self.tracker = tracker

    def rollback_to(self, target_hash: str,
                    reason: str = "") -> PromptVersion | None:
        """Restore a previous prompt version as the active one."""
        target = self.store.get_by_hash(target_hash)
        if not target:
            return None

        # Find current "active" version (most recent)
        versions = self.store.list_versions(limit=1)
        if versions and versions[0].prompt_hash == target_hash:
            # Already active
            return target

        # Re-save as a new version with rollback metadata
        new_version = self.store.save(
            content=target.content,
            model=target.model,
            tags=target.tags + ["rollback"],
            label=f"rollback to {target.label or target.prompt_hash[:8]}",
            parent_hash=versions[0].prompt_hash if versions else "",
            metadata={
                "rollback_from": versions[0].prompt_hash if versions else "",
                "rollback_to": target_hash,
                "reason": reason,
            },
        )

        return new_version

    def rollback_n(self, n: int = 1, reason: str = "") -> PromptVersion | None:
        """Roll back N versions from current."""
        versions = self.store.list_versions(limit=n + 1)
        if len(versions) <= n:
            return None

        target = versions[n]  # N versions back
        return self.rollback_to(target.prompt_hash, reason)

    def pin_version(self, prompt_hash: str) -> bool:
        """Pin a version — prevents auto-rollback from touching it."""
        self.store.conn.execute("""
            UPDATE prompt_versions SET is_pinned = 1
            WHERE prompt_hash = ?
        """, (prompt_hash,))
        self.store.conn.commit()
        return True

Putting It All Together: The Complete Workflow

# --- Setup ---
store = PromptVersionStore("prompts.db")
differ = PromptDiffer()
tracker = ImpactTracker(store)
rollback = PromptRollback(store, tracker)

# --- Version 1: Initial system prompt ---
v1_content = """## Role
You are a helpful coding assistant.

## Instructions
- Answer concisely
- Show code examples
- Use type hints"""

v1 = store.save(content=v1_content, model="gpt-4o",
                tags=["coding", "concise"], label="v1-baseline")

# --- Version 2: Add chain-of-thought ---
v2_content = """## Role
You are a helpful coding assistant.

## Instructions
- Think step by step before answering
- Answer concisely
- Show code examples with type hints
- Explain your reasoning"""

v2 = store.save(content=v2_content, model="gpt-4o",
                tags=["coding", "cot"], label="v2-cot",
                parent_hash=v1.prompt_hash)

# --- Diff them ---
changes = differ.token_diff(v1.content, v2.content)
summary = differ.summarize_diff(changes)
print(f"Token diff: +{summary['additions']} -{summary['deletions']}")

sections = differ.section_diff(v1.content, v2.content)
for s in sections:
    print(f"  Section '{s['section']}': {s['status']}")

semantic = differ.semantic_diff(v1.content, v2.content)
print(f"Semantic: +{semantic['added_words']} -{semantic['removed_words']} words")
print(f"  Instruction changes: {semantic['net_instruction_change']}")

# --- Track impact over 100 turns ---
import random
for i in range(100):
    # v1: baseline performance
    tracker.record(v1.prompt_hash, {
        "accuracy": random.gauss(0.72, 0.08),
        "tokens_output": random.gauss(340, 50),
        "latency_ms": random.gauss(1200, 200),
    })
    # v2: CoT — higher accuracy, but more tokens
    tracker.record(v2.prompt_hash, {
        "accuracy": random.gauss(0.81, 0.06),
        "tokens_output": random.gauss(520, 80),
        "latency_ms": random.gauss(1850, 250),
    })

# --- Compare ---
comparison = tracker.compare(v1.prompt_hash, v2.prompt_hash)
print("\n--- Impact Comparison ---")
for metric, data in comparison.items():
    arrow = "↑" if data["delta"] > 0 else "↓"
    print(f"  {metric:20s} {data['version_a']:.2f} → {data['version_b']:.2f} "
          f"({arrow}{abs(data['delta_pct']):.1f}%)")

# --- Rollback if v2 is too expensive ---
if comparison.get("tokens_output", {}).get("delta_pct", 0) > 40:
    print("\nToken increase >40%, rolling back...")
    restored = rollback.rollback_to(v1.prompt_hash,
                                    reason="v2 CoT too token-heavy")
    print(f"Restored: {restored.label}")

The Decision Matrix

When comparing two prompt versions, use this matrix to decide:

MetricGood SignBad SignThreshold
Accuracy↑ 5%+↓ 3%+3% = significant
Tokens output↓ or flat↑ 40%+40% = cost alarm
Latency↓ or flat↑ 30%+30% = UX alarm
Error rate↓↑ 2x2x = critical
Tool misuse↓↑Any increase = investigate
Statistical significance: Don't compare with under 30 samples per version. With 30+ samples, a 5% accuracy difference is usually real. With under 10, you're reading noise.

Common Pitfalls

Pitfall 1: Versioning composed prompts wrong. If your system prompt is assembled from templates + context + skills, hash the final composed string, not the template. Two different template combinations can produce the same output.
Pitfall 2: Too many micro-versions. If every tiny edit creates a new version, you drown in noise. Use semantic tags to group versions — "experiment", "production", "rollback".
Pitfall 3: Ignoring model interactions. The same prompt performs differently on GPT-4o vs Claude vs Gemini. Always record the model with each version. Comparing a GPT-4o result to a Claude result with the "same" prompt is meaningless.

Conclusion

Prompt versioning answers four questions that every agent team eventually faces:

  1. What changed? → Diff engine (token, section, semantic)
  2. When? → Version store with timestamps
  3. Was it better? → Impact tracker with statistical comparison
  4. Can we go back? → Rollback with chain-of-versions

The full implementation is 350 lines of pure Python — SQLite for storage, difflib for diffs, hashlib for dedup. No external dependencies, no cloud services, no complexity tax.

Stop guessing whether your prompt changes help or hurt. Version them, measure them, and let the data decide.
