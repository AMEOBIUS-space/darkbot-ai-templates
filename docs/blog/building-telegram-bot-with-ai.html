Building a Telegram Bot with AI Integration

A practical guide using aiogram 3, OpenAI, and FSM for production bots.

Why aiogram 3?

aiogram 3 is the most modern async Telegram bot framework for Python. It supports:

  • Finite State Machine (FSM) for conversation flows
  • Inline keyboards for interactive UI
  • Middleware for rate limiting, logging
  • Full async/await support

Quick Start

pip install aiogram openai python-dotenv

1. Bot Setup

from aiogram import Bot, Dispatcher
from aiogram.filters import Command
from aiogram.types import Message

bot = Bot(token="YOUR_TOKEN")
dp = Dispatcher()

@dp.message(Command("start"))
async def start(msg: Message):
    await msg.answer("Hello! I'm an AI bot. Ask me anything!")

if __name__ == "__main__":
    import asyncio
    asyncio.run(dp.start_polling(bot))

2. Add AI Integration

from openai import OpenAI
client = OpenAI(api_key="YOUR_KEY")

@dp.message()
async def ai_reply(msg: Message):
    response = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": msg.text}]
    )
    await msg.answer(response.choices[0].message.content)

3. FSM for Multi-step Conversations

class Form(StatesGroup):
    name = State()
    question = State()

@dp.message(Command("ask"))
async def ask_start(msg: Message, state: FSMContext):
    await state.set_state(Form.name)
    await msg.answer("What's your name?")
