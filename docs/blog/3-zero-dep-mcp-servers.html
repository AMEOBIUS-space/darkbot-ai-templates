The Problem

Most MCP servers for web scraping, crypto monitoring, and email wrap heavy libraries: Playwright (190MB), Selenium (WebDriver binary), requests (unnecessary dependency), npm/Puppeteer (Node.js runtime in a Python project). What if you could give your AI agent 30 tools with nothing but Python's standard library?

>

The Solution

Three MCP servers using only Python stdlib (socket, urllib, smtplib, json, re, struct, base64). No pip installs. No npm. No API keys (for crypto and email). Just Python 3.10+.

1. mcp-cdp-scraper — Web Scraping via CDP

Connects to any Chrome/Chromium instance via raw WebSocket CDP commands. No Playwright, no Selenium.

>

The WebSocket Trick

Chrome's DevTools Protocol speaks WebSocket, and Python's socketmodule can implement RFC 6455 from scratch in ~100 lines:

class CDPConnection:
 def connect(self):
 host, port, path = self._parse_ws_url()
 self.sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
 self.sock.connect((host, port))
 key = base64.b64encode(os.urandom(16)).decode()
 handshake = f"GET {path} HTTP/1.1\r\n..."
 self.sock.sendall(handshake.encode())

 def send_command(self, method, params):
 self.msg_id += 1
 payload = json.dumps({"id": self.msg_id, "method": method, "params": params})
 self._send_ws_frame(payload)
 return self._recv_response()

12 Tools

scrape_page(navigate + extract everything), extract_text, extract_links, extract_images, extract_table, fill_form(React/Vue compatible), click_element, screenshot, get_html, wait_for, scroll_to, list_tabs.

>

React/Vue Form Filling

Direct element.value = "x"doesn't work in React/Vue. The solution uses native setters + synthetic events:

const setter = Object.getOwnPropertyDescriptor(
 HTMLInputElement.prototype, "value"
).set
setter.call(el, "value")
el.dispatchEvent(new Event("input", {bubbles: true}))
el.dispatchEvent(new Event("change", {bubbles: true}))

3. mcp-email-sender — Validation & SMTP

Four validation layers: format (RFC 5322 regex), disposable (16 domains), role-based (admin@, info@), MX records (DNS lookup). SMTP via stdlib smtplibwith TLS/SSL.
