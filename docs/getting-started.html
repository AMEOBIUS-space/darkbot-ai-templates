Getting Started

New to DarkBot AI? Follow these 5 steps.

>

Step 1: Install

# One-liner install
curl -sL https://raw.githubusercontent.com/AMEOBIUS/darkbot-ai-templates/main/install.sh | bash

# Or clone manually
git clone https://github.com/AMEOBIUS/darkbot-ai-templates.git
cd darkbot-ai-templates
>

Step 2: Try Demos (no API keys!)

python tg-bot-template/demo.py # TG Bot simulation
python crypto-trading-bot/demo.py # Live BTC price!
python web-scraper-template/demo.py # Scraping demo
python fastapi-template/demo.py # API simulation
python cdp-automation-template/demo.py # CDP simulation

Or use Make: cd tg-bot-template && make demo

>

Step 3: Choose a Template

Browse by categoryor check use cases.

Each template: $39-$99 | Bundle: $199

>

Step 4: Customize

cd tg-bot-template
cp .env.example .env # Add your tokens
# Edit bot.py for your needs
make run

Need help customizing? Contact @darkbot_ai_bot

>

Step 5: Deploy

# Docker
docker-compose up -d # For TG Bot / FastAPI

# Or manual
pip install -r requirements.txt
python bot.py
>
