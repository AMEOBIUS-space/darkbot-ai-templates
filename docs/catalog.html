How to Use Any Server

# 1. Clone
git clone https://github.com/aaameobius-crypto/mcp-context-guard.git

# 2. No dependencies to install!

# 3. Run tests
python -m pytest tests/ -v

# 4. Start the server (stdio JSON-RPC)
python src/server.py

Why Zero Dependencies?

Every dependency is a liability — supply chain attack surface, version conflicts, broken installs in CI. Python's stdlib has json, sqlite3, hashlib, http.server, re, collections, pathlib, urllib. That covers 95% of what an MCP server needs.

The 5% that needs external libraries (heavy NLP, ML inference) doesn't belong in a utility MCP server anyway — those belong in dedicated model-serving infrastructure.
