# Tor Market Scraper Template — .onion Marketplace Monitoring

Production-ready .onion marketplace scraper:
- Tor SOCKS5 proxy support (aiohttp-socks)
- Retry logic with backoff
- Product listing extraction (HTML + JSON-LD)
- Price monitoring (continuous mode)
- Pagination support
- Rate limiting (3s between requests)
- JSON and CSV export
- Customizable extraction patterns per marketplace

## Quick Start
```bash
pip install -r requirements.txt
# Start Tor: tor --SocksPort 9050
python scraper.py
```
